AEDAT tools is a library of functions for importing, exporting, converting visualising and otherwise working with .aedat files. 

The library is implemented in both Matlab and Python. 

The library is documented at:

www.inilabs.com/support/software/aedattools

2016_07_05: Since this library is work in progress, the above may not yet be true. 
If you need this functionality and it hasn't been written yet, please let us know at support@inilabs.com, since this helps us prioritise our work.
